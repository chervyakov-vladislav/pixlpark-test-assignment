### Запуск проекта локально
```
git clone https://github.com/chervyakov-vladislav/pixlpark-test-assignment.git .
cd server -> yarn -> yarn
cd ../client -> yarn -> yarn start
```

Ввадите localhost:3000 в вашем браузере
